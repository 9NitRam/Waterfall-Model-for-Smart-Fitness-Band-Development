# Waterfall Model for Smart Fitness Band Development

## Project Details

* Course Name: Critical Thinking, Design Thinking, Leadership and Teamwork
* College Name: GALGOTIAS UNIVERSITY
* Batch Number: Batch No. 5
* Task Number: Task #03

## Description

This project demonstrates the Waterfall Software Development Model for a startup developing a Smart Fitness Band. The model includes the following phases:

1. Requirement Analysis
2. System Design
3. Implementation
4. Testing
5. Deployment
6. Maintenance

## Conclusion

The Waterfall model is a sequential software development process where each phase must be completed before moving to the next phase. It is suitable for projects with clear and fixed requirements.
